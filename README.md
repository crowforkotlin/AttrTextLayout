- 这是一个属性文本布局控件，仅支持原生使用，能实现以下功能
- 支持各种文本的特效、例如四个方向移动、擦除、百叶窗、跑马灯、等特效
- 支持多行（自动、手动换行）、单行、更新文本时继续执行特效
- 支持配置多种文本策略
- 毫秒级绘制处理，最大支持200个组件异步执行特效（略有卡顿）稳定50个
